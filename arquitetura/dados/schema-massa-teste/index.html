    Schema e massa de laboratório (MySQL 8+)¶

    Recomendado para leitura prévia

    • Pacote experimental — ordem de reprodução (schema, bateria, gabarito, baseline).
    • Banco laboratorial — domínios de negócio e entidades-âncora.
    • Carga no cluster — pipeline de ingestão no Hive/Atlas.
    • Catálogo Atlas — camada semântica no cluster.

    Decisão¶

    A massa de avaliação corresponde ao schema relacional base_laboratorial, versionado em Dump MySQL base_laboratorial (MySQL 8+). O inventário abaixo é derivado automaticamente desse dump.

    Download e importação¶

    mysql -u root -p < dados/base_laboratorial.sql

    Requisitos e convenções: Instruções de importação MySQL.

    Regras da massa¶

    • O schema é fixado antes das corridas finais e versionado em datasetVersion.
    • Alteração do schema exposto exige nova versão e ADR.
    • Dados sensíveis são mascarados na carga conforme Carga no cluster.
